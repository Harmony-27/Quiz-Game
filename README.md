# Quiz Game Project

This is a simple Python project that works as a quiz game. It asks multiple-choice questions to the user and checks their answers.

## Project Description

This program presents a set of questions with four options (A, B, C, D).
The user selects an answer for each question.

After each answer:

* The program tells whether it is correct or wrong.
* The score is updated.

At the end of the quiz:

* The total score is displayed.
* A performance message is shown based on the score.

## Features

* Multiple-choice quiz system
* Takes input from the user
* Checks answers instantly
* Keeps track of score
* Displays correct answer if the user is wrong
* Shows final result with performance message

## How It Works

1. The quiz starts with a welcome message
2. Questions are asked one by one
3. User enters an answer (A/B/C/D)
4. Score updates based on correct answers
5. Final score is displayed at the end
